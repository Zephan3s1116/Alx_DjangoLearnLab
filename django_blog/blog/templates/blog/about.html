Welcome to Our Blog Platform

This is a Django-based blogging platform created as part of the ALX Django Learning Lab. The blog demonstrates the fundamental concepts of Django web development, including:

  • Database models and relationships
  • Template inheritance and static files
  • Django admin interface
  • URL routing and views

Features

  • Create and manage blog posts
  • User authentication and authorization
  • Responsive design with Bootstrap
  • Clean and intuitive interface

Technology Stack

  • Backend: Django (Python)
  • Database: SQLite
  • Frontend: HTML, CSS, JavaScript, Bootstrap
